# PPT-Master 使用指南 — 摘要

> AI 驱动的 PPT 自动生成系统，通过多角色协作将源文档转换为原生可编辑的 PPTX 文件。

## 核心流程

```
源文档 → 创建项目 → [模板] → 策略师确认 → [图片获取] → 执行器生成SVG → 后处理 → 导出PPTX
```

---

## 关键要点

1. **支持多种输入格式**：PDF、DOCX、PPTX、Excel、网页 URL、EPUB、HTML、Markdown，甚至无材料的话题（自动触发网络调研）。
2. **唯一硬性确认点**：策略师阶段会提出 8 项设计建议（画布格式、页数、受众、风格、配色、图标、字体、图片策略），用户确认后全流程自动执行。
3. **SVG 中间层**：执行器逐页生成 SVG，经过质量检查和后处理（三个不可合并的脚本）最终导出为原生 PPTX。
4. **18 套内置模板**：覆盖学术、企业、政府、科技等场景，按完整路径指定使用。
5. **串行流水线**：步骤不可跳跃，禁止跨阶段打包，执行器不可委托子代理。
